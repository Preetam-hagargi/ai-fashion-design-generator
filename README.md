\# AI Fashion Design Generator



AI-powered system that generates fashion print designs for hoodies, t-shirts, and jackets using Stable Diffusion.



\## Tech Stack



\- FastAPI (backend via Colab)

\- Stable Diffusion

\- React.js frontend

\- Ngrok for tunneling



\## How to Run Backend



Open the notebook in Google Colab and run all cells.



\## How to Run Frontend



cd frontend  

npm install  

npm start
